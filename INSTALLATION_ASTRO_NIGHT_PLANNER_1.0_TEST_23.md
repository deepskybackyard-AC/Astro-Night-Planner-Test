# Installation Astro Night Planner 1.0 Test 23

Diese ZIP-Datei ist die Testversion 23 für das Test-Repository.

## Installation

1. ZIP-Datei entpacken.
2. Inhalt vollständig in das Test-Repository hochladen.
3. Vorhandene Dateien überschreiben.
4. GitHub Pages Deployment abwarten.
5. Die Test-App im Browser öffnen und bei Bedarf einmal hart neu laden.

## Cache-Hinweis

Die Service-Worker-Cache-Version wurde auf `1.0.0-test.23` erhöht. Alte Test-Caches werden beim Aktivieren des neuen Service Workers entfernt.

## Wichtige Änderungen

- Verbesserte Mondanzeige in der Aladin-Objektkarte als pixelstabiles SVG/DOM-Symbol.
- Exakte Kopplung von Mondposition, Symbol und Beschriftung an dieselbe berechnete RA/Dec-Koordinate.
- Beleuchtete Mondfläche weiß bis leicht warmweiß, unbeleuchtete Seite transparent bzw. dunkel-transparent.
- Neuer Button `Basisfilter zurücksetzen` neben `Filter anwenden`.
- Hilfe und PDF-Handbuch DE/EN aktualisiert.
