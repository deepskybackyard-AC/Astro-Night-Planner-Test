# Installation Astro Night Planner 1.0.1-test.1

Dies ist eine Testversion für das Test-Repository.

## Einspielen

1. ZIP entpacken.
2. Inhalt in das Test-Repository kopieren.
3. Commit und Push auf `main`.
4. GitHub Pages Deploy abwarten.
5. Im Browser hart neu laden oder Websitedaten des Test-Repositorys löschen, falls alte Caches sichtbar bleiben.

## Wichtige Änderungen

- Hauptnavigation `Planung` / `Einstellungen` in die feste Titelzeile verschoben.
- Untere/floating Navigation entfernt.
- LDN-Katalog mit 1787 benannten Einträgen ergänzt.
- Lokaler Katalogcache auf `catalog-v11` erhöht.

## Prüfung nach Deployment

- `VERSION.json` muss `1.0.1-test.1` anzeigen.
- Suche nach `LDN 1093` und `LDN1093` testen.
- Katalogfilter `LDN/LBN` aktivieren und Treffer prüfen.
- Desktop und Mobilansicht der neuen Header-Navigation prüfen.
