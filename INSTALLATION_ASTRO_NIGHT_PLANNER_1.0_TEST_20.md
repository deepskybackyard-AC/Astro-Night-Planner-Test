# Installation Astro Night Planner 1.0.0-test.20

1. ZIP entpacken.
2. Den kompletten Inhalt in das Test-Repository hochladen.
3. Vorhandene Dateien ersetzen.
4. Nach dem Deployment die installierte PWA bzw. den Browser vollständig neu laden.
5. Unter Info prüfen, dass Version `1.0.0-test.20` angezeigt wird.

Hinweis: Wenn noch eine alte Oberfläche erscheint, den Updatehinweis bestätigen, die PWA schließen und erneut öffnen. Die Testversion nutzt weiterhin die getrennte Test-Datenbank `astro-night-planner-test-v1`.
