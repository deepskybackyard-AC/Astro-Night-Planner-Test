Local surveys - operating-system guides

Local HiPS surveys are served through a local HTTP address, not through file paths. The Astro Night Planner settings are the same on all desktop systems; only the helper program differs by operating system.

Common principle

The Local Survey Server serves one shared survey root folder. In Astro Night Planner the base URL is normally http://127.0.0.1:8765/. Each survey uses a relative path, for example nsns/ohs8/.

~/AstroSurveys/
  nsns/
    ohs8/
      properties
      Allsky...
      Norder...
    halpha8/
    oiii8/

Important

The web app cannot directly start local programs. The Open Local Survey Server button can only open the local configuration interface when the server is already running. If the server is not reachable, start the matching helper manually or enable autostart.

Astro Night Planner - Local Survey Server documentation

Downloading another HiPS survey

Select Other / custom HiPS survey in the download manager. Enter the complete HiPS source URL and a relative target path inside the survey root, for example custom/my-survey/.

A complete recursive download is possible only when the source server provides a properties file and MOC coverage map and the download is technically and legally permitted. Very large surveys may require substantial storage and download time.

Windows configuration from version 1.0

On first start, server 1.1 (Build 1.1.5) searches for the previous ANP-Local-Survey-Server-config.json in the program folder, current folder, user folder and AppData. If found, survey root, port, language and autostart settings are migrated to the new configuration. The old file is retained as a backup.

Windows tray controls

The tray icon context menu provides Open interface, Start server, Stop server and Exit. Build 1.1.5 restores the icon after Explorer restarts and includes a safety fallback for reliable exit without Task Manager.

Selecting relative paths in Planner

Planner displays HiPS folders detected by the running server in a dedicated dialog with clear folder names and the survey root. A numeric prompt is no longer used.
