Linux/macOS: nativer Local Survey Server 1.1

Diese Variante ist die zusätzliche Lösung für Linux und macOS. Sie ersetzt nicht das Windows-Programm und benötigt weder Wine noch Python.

Was das Programm macht

  • Es stellt einen gemeinsamen Survey-Hauptordner per lokalem HTTP-Server bereit.
  • Standard-Survey-URL: http://127.0.0.1:8765/
  • Standard-Konfigurationsoberfläche: http://127.0.0.1:8776/
  • Die Oberfläche läuft im Browser und funktioniert unter Linux und macOS gleich.

Welches Binary verwenden?

SystemBinary
Linux Intel/AMD 64-bitbin/anp-local-survey-server-linux-amd64
Linux ARM64bin/anp-local-survey-server-linux-arm64
macOS Intelbin/anp-local-survey-server-darwin-amd64
macOS Apple Siliconbin/anp-local-survey-server-darwin-arm64

Start unter Linux

  1. Paket entpacken.
  2. Passendes Binary ausführbar machen.
  3. Mit Survey-Hauptordner starten.
chmod +x bin/anp-local-survey-server-linux-amd64
./bin/anp-local-survey-server-linux-amd64 --root "$HOME/AstroSurveys" --port 8765 --open

Start unter macOS

  1. Paket entpacken.
  2. Passendes Binary ausführbar machen.
  3. Beim ersten Start kann macOS eine Sicherheitswarnung zeigen. Dann im Finder oder in Datenschutz & Sicherheit den Start erlauben.
  4. Mit Survey-Hauptordner starten.
chmod +x bin/anp-local-survey-server-darwin-arm64
./bin/anp-local-survey-server-darwin-arm64 --root "$HOME/AstroSurveys" --port 8765 --open

Eintrag im Astro Night Planner

Lokale Survey-Basis-URL:
http://127.0.0.1:8765/

Relativer Pfad für NSNS OHS8:
nsns/ohs8/

Die Datei muss im Browser erreichbar sein: http://127.0.0.1:8765/nsns/ohs8/properties.

Autostart und Stoppen

Linux: siehe systemd-user-Anleitung. macOS: siehe LaunchAgent-Anleitung. Im Terminal kann der Server mit Strg+C gestoppt werden. In der Browseroberfläche gibt es zusätzlich Programm beenden.

Ordnerauswahl und Survey-Download

Hauptordner und Downloadziel können über einen Betriebssystem-Ordnerdialog gewählt werden. Bekannte NSNS-Surveys lassen sich in der Browseroberfläche herunterladen, pausieren, fortsetzen, abbrechen und prüfen.
