Survey-Daten herunterladen und verwalten

ANP Local Survey Server 1.1 kann alle in der App eingebauten Surveys mit vorbelegten Quelladressen sowie weitere kompatible HiPS-Surveys direkt in den konfigurierten Survey-Hauptordner herunterladen. Die Funktion steht in der Windows- sowie in der Linux/macOS-Variante zur Verfügung.

Downloadverwaltung öffnen

  1. Local Survey Server starten.
  2. Im Astro Night Planner unter Lokale Survey-Quellen auf Survey-Daten herunterladen und verwalten klicken.
  3. Alternativ die Serveroberfläche öffnen und zum Abschnitt Survey-Daten herunterladen und verwalten wechseln.

Survey und Zielordner

Wähle einen bekannten NSNS-Survey. Online-Quelladresse und empfohlener relativer Zielpfad werden vorbelegt. Der Zielordner muss innerhalb des Survey-Hauptordners liegen.

Survey-Hauptordner: D:\AstroSurveys
Relativer Zielpfad: nsns/ohs8/
Ergebnis: D:\AstroSurveys
sns\ohs8\

Mit Zielordner auswählen öffnet das Serverprogramm einen Betriebssystem-Ordnerdialog. Unter Linux wird dafür zenity oder kdialog verwendet, unter macOS der native Finder-Dialog.

Fortschritt und Wiederaufnahme

  • Download starten: liest das Verzeichnislisting des Quellservers und lädt Ordner und Dateien rekursiv.
  • Pausieren/Fortsetzen: hält den aktiven Download an beziehungsweise setzt ihn fort.
  • Abbrechen: beendet den aktuellen Lauf. Teilweise geladene Dateien bleiben als .part erhalten und können beim nächsten Start fortgesetzt werden.
  • Bereits vollständig vorhandene Dateien werden übersprungen.
  • Fortschritt, übertragene Datenmenge, gefundene und abgeschlossene Dateien werden angezeigt.

Vorhandene Daten prüfen

Vorhandene Daten prüfen kontrolliert insbesondere die Datei properties, das angegebene Kachelformat und die Zahl vorhandener Dateien. Die Prüfung ersetzt keine kryptografische Prüfsumme des gesamten externen Datenbestands, erkennt aber eine fehlende oder offensichtlich unvollständige HiPS-Struktur.

Voraussetzungen und Grenzen

Der Online-Quellserver muss ein lesbares Verzeichnislisting bereitstellen. Die bekannten NSNS-Adressen sind vorbelegt, können bei Bedarf aber bearbeitet werden. Survey-Daten können sehr groß sein; ausreichend freier Speicher und eine stabile Internetverbindung sind erforderlich.

Andere HiPS-Surveys herunterladen

Wähle in der Downloadverwaltung Anderer / benutzerdefinierter HiPS-Survey. Trage anschließend die vollständige HiPS-Quelladresse und einen relativen Zielpfad innerhalb des Survey-Hauptordners ein, zum Beispiel custom/mein-survey/.

Ein vollständiger rekursiver Download ist nur möglich, wenn der Quellserver ein Verzeichnislisting bereitstellt und der Download technisch sowie rechtlich zulässig ist. Sehr große Surveys können erheblichen Speicherplatz und lange Downloadzeiten benötigen.

Windows-Konfiguration aus Version 1.0

Server 1.1 sucht beim ersten Start nach der bisherigen Datei ANP-Local-Survey-Server-config.json im Programmordner, aktuellen Ordner, Benutzerordner und AppData. Wird sie gefunden, werden Survey-Hauptordner, Port, Sprache und Autostart-Einstellungen in die neue Konfiguration übernommen. Die alte Datei bleibt als Sicherung erhalten.

Tray-Steuerung unter Windows

Das Rechtsklickmenü des Tray-Symbols bietet zuverlässig Oberfläche öffnen, Server starten, Server stoppen und Beenden. Server stoppen beendet nur den Survey-Dateiserver. Beenden schließt auch Browseroberfläche und Tray vollständig.

Relative Pfade im Planner auswählen

Der Planner zeigt die vom laufenden Server gefundenen HiPS-Ordner in einem eigenen Dialog mit verständlichen Ordnernamen und dem Survey-Hauptordner an. Eine Nummerneingabe oder ein Browser-Prompt ist nicht mehr erforderlich.
