macOS autostart with LaunchAgent

A LaunchAgent starts the Local Survey Server automatically after user login.

1. Copy the plist

mkdir -p ~/Library/LaunchAgents
cp templates/de.deepskyastrophoto.anp-local-survey-server.plist   ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist

2. Adjust paths

Adjust the path to the binary and the survey root folder in the plist. Apple Silicon Macs normally use bin/anp-local-survey-server-darwin-arm64, Intel Macs use bin/anp-local-survey-server-darwin-amd64.

3. Load and start

launchctl load ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist
launchctl start de.deepskyastrophoto.anp-local-survey-server

Stop and remove

launchctl stop de.deepskyastrophoto.anp-local-survey-server
launchctl unload ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist

macOS security

macOS may require a security approval for unsigned programs on first launch. Allow the launch in Finder or in Privacy & Security.
