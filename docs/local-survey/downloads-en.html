Download and manage survey data

ANP Local Survey Server 1.1.2 can download known NSNS HiPS surveys and other compatible HiPS surveys directly into the configured survey root. The feature is available in both the Windows and Linux/macOS editions.

Open the download manager

  1. Start Local Survey Server.
  2. In Astro Night Planner, open Local survey sources and click Download and manage survey data.
  3. Alternatively open the server interface and go to Download and manage survey data.

Survey and target folder

Select a known NSNS survey. The online source URL and recommended relative target path are filled in. The target folder must be inside the survey root.

Survey root: /home/user/AstroSurveys
Relative target path: nsns/ohs8/
Result: /home/user/AstroSurveys/nsns/ohs8/

Choose target folder opens an operating-system folder dialog. Linux uses zenity or kdialog; macOS uses the native Finder dialog.

Progress and resume

  • Start download: reads the source directory listing and downloads folders and files recursively.
  • Pause/Resume: pauses or resumes the active download.
  • Cancel: ends the current run. Partial files remain as .part and can be resumed later.
  • Complete existing files are skipped.
  • Progress, transferred bytes, discovered files and completed files are displayed.

Validate existing data

Validate existing data checks the properties file, declared tile format and number of files. It is not a cryptographic checksum of the complete external data set, but detects a missing or obviously incomplete HiPS structure.

Requirements and limitations

The online source server must provide a readable directory listing. Known NSNS addresses are preconfigured but remain editable. Survey data can be very large; sufficient free storage and a stable internet connection are required.

Downloading another HiPS survey

Select Other / custom HiPS survey in the download manager. Enter the complete HiPS source URL and a relative target path inside the survey root, for example custom/my-survey/.

A complete recursive download is possible only when the source server provides a directory listing and the download is technically and legally permitted. Very large surveys may require substantial storage and download time.

Windows configuration from version 1.0

On first start, server 1.1.2 searches for the previous ANP-Local-Survey-Server-config.json in the program folder, current folder, user folder and AppData. If found, survey root, port, language and autostart settings are migrated to the new configuration. The old file is retained as a backup.
