Download and manage survey data

ANP Local Survey Server 1.1 (Build 1.1.5) can download all 15 surveys built into the app using verified default service URLs, plus other compatible HiPS surveys, directly into the configured survey root. The same download logic is available on Windows, Linux and macOS.

How downloading works

  1. Select a survey. The source and recommended relative target path are prefilled.
  2. Check download source loads properties and Moc.fits and reports a concrete HTTP error if the source is unavailable.
  3. The server reads tile format and maximum order from properties.
  4. Required Norder/Dir/Npix tiles are derived from the MOC coverage map. A web-directory listing is not required.
  5. Missing target folders are created automatically.

Survey root and relative target path

Survey root:          /home/user/AstroSurveys
Relative target path: nsns/halpha8/
Resolved target:      /home/user/AstroSurveys/nsns/halpha8/

A Windows root uses backslashes while the relative HiPS path uses forward slashes. The server normalizes both for the operating system.

Save source and target path stores both values per survey. The button briefly shows Saved ✓; no success popup is used.

Progress, resume and interrupted connections

  • Start download: begins the MOC-based download.
  • Pause/Resume: pauses or resumes the active download.
  • Cancel: ends the run. Partial files remain as .part.
  • Complete existing files are skipped.
  • Temporary network errors, timeouts and HTTP 429/5xx are retried automatically for each file with increasing delays.
  • One failed tile no longer stops the complete download.
  • After the main pass, remaining files are retried automatically in a recovery pass.
  • Downloads from www.simg.de use two parallel connections; other sources use four by default.
  • Status shows retry attempt, wait time, failed files and parallel connections.

Other HiPS surveys

Select Other / custom HiPS survey and enter the complete service URL and a relative target path. A valid properties file and Moc.fits are required. Downloading must be technically and legally permitted.

Storage requirements

HiPS surveys can require many gigabytes or considerably more. The estimate in properties is only an indication. Ensure sufficient free storage and a stable connection.
