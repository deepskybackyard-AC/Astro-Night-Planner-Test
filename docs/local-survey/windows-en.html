Windows: ANP Local Survey Server 1.1.1

The Windows helper is the recommended variant for Windows users. Double-clicking it automatically opens the browser interface, while the tray menu, autostart and exit function remain available.

Start

  1. Extract ANP-Local-Survey-Server-1.1.1.zip.
  2. Double-click ANP-Local-Survey-Server-1.1.1.exe.
  3. The browser interface opens automatically.
  4. Check survey root folder and port.
  5. Use Start server to start the local survey file server.

Browser interface and tray

Closing the browser window does not exit the program. The server and tray icon keep running. Double-click the tray icon to reopen the browser interface. The tray menu provides Open interface, Start server, Stop server and Exit. Only Exit fully stops the helper and releases the EXE file.

Astro Night Planner settings

Local survey base URL:
http://127.0.0.1:8765/

Relative path for NSNS OHS8:
nsns/ohs8/

Check in the browser: http://127.0.0.1:8765/nsns/ohs8/properties.

Autostart

Enable Start with Windows in the Windows interface if the Local Survey Server should be available automatically after Windows login.

Folder selection and survey downloads

The root folder and download target can be selected with an operating-system folder dialog. Known NSNS surveys can be downloaded, paused, resumed, cancelled and validated in the browser interface.
