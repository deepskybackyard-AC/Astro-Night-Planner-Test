macOS-Autostart mit LaunchAgent

Ein LaunchAgent startet den Local Survey Server automatisch beim Benutzer-Login.

1. plist kopieren

mkdir -p ~/Library/LaunchAgents
cp templates/de.deepskyastrophoto.anp-local-survey-server.plist   ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist

2. Pfade anpassen

Passe in der plist den Pfad zum Binary und den Survey-Hauptordner an. Für Apple Silicon wird normalerweise bin/anp-local-survey-server-darwin-arm64 verwendet, für Intel-Macs bin/anp-local-survey-server-darwin-amd64.

3. Laden und starten

launchctl load ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist
launchctl start de.deepskyastrophoto.anp-local-survey-server

Stoppen und entfernen

launchctl stop de.deepskyastrophoto.anp-local-survey-server
launchctl unload ~/Library/LaunchAgents/de.deepskyastrophoto.anp-local-survey-server.plist

macOS-Sicherheit

macOS kann beim ersten Start nicht signierter Programme eine Sicherheitsfreigabe verlangen. Dann den Start im Finder oder in Datenschutz & Sicherheit erlauben.
