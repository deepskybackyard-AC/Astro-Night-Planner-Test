Linux autostart with systemd --user

A systemd user service can start the Local Survey Server automatically in the background after login.

1. Copy the service file

mkdir -p ~/.config/systemd/user
cp templates/anp-local-survey-server.service   ~/.config/systemd/user/anp-local-survey-server.service

2. Adjust ExecStart

Open the file and adjust the binary path and survey root folder.

nano ~/.config/systemd/user/anp-local-survey-server.service
ExecStart=/home/andreas/ANP-Local-Survey-Server-Linux-macOS-1.0/bin/anp-local-survey-server-linux-amd64 --root /home/andreas/AstroSurveys --port 8765 --no-open

3. Enable the service

systemctl --user daemon-reload
systemctl --user enable --now anp-local-survey-server.service

Status, stop, disable

systemctl --user status anp-local-survey-server.service
systemctl --user stop anp-local-survey-server.service
systemctl --user disable anp-local-survey-server.service
