Windows: ANP Local Survey Server 1.1.3

Das Windows-Hilfsprogramm ist die empfohlene Variante für Windows-Nutzer. Beim Doppelklick öffnet es automatisch die Browseroberfläche und bleibt danach über das Tray-Menü, Autostart und die Beenden-Funktion erreichbar.

Start

  1. Paket ANP-Local-Survey-Server-1.1.3.zip entpacken.
  2. ANP-Local-Survey-Server-1.1.3.exe per Doppelklick starten.
  3. Die Browseroberfläche öffnet sich automatisch.
  4. Survey-Hauptordner und Port prüfen.
  5. Mit Server starten den lokalen Survey-Dateiserver starten.

Browseroberfläche und Tray

Das Schließen des Browserfensters beendet das Programm nicht. Der Server und das Tray-Symbol laufen weiter. Ein Doppelklick auf das Tray-Symbol öffnet die Browseroberfläche erneut. Das Tray-Menü bietet Oberfläche öffnen, Server starten, Server stoppen und Beenden. Nur Beenden stoppt das Hilfsprogramm vollständig und gibt die EXE frei.

Eintrag im Planner

Lokale Survey-Basis-URL:
http://127.0.0.1:8765/

Relativer Pfad für NSNS OHS8:
nsns/ohs8/

Prüfe im Browser: http://127.0.0.1:8765/nsns/ohs8/properties.

Autostart

Aktiviere in der Windows-Oberfläche Mit Windows starten, wenn der Local Survey Server beim Windows-Start automatisch verfügbar sein soll.

Ordnerauswahl und Survey-Download

Hauptordner und Downloadziel können über einen Betriebssystem-Ordnerdialog gewählt werden. Alle in der App eingebauten Surveys sind mit Standard-Quelladressen aufgeführt und können in der Browseroberfläche heruntergeladen, pausiert, fortgesetzt, abgebrochen und geprüft werden.

Download-Anleitung öffnen
Astro Night Planner - Local Survey Server documentation

Andere HiPS-Surveys herunterladen

Wähle in der Downloadverwaltung Anderer / benutzerdefinierter HiPS-Survey. Trage anschließend die vollständige HiPS-Quelladresse und einen relativen Zielpfad innerhalb des Survey-Hauptordners ein, zum Beispiel custom/mein-survey/.

Ein vollständiger rekursiver Download ist nur möglich, wenn der Quellserver ein Verzeichnislisting bereitstellt und der Download technisch sowie rechtlich zulässig ist. Sehr große Surveys können erheblichen Speicherplatz und lange Downloadzeiten benötigen.

Windows-Konfiguration aus Version 1.0

Server 1.1.3 sucht beim ersten Start nach der bisherigen Datei ANP-Local-Survey-Server-config.json im Programmordner, aktuellen Ordner, Benutzerordner und AppData. Wird sie gefunden, werden Survey-Hauptordner, Port, Sprache und Autostart-Einstellungen in die neue Konfiguration übernommen. Die alte Datei bleibt als Sicherung erhalten.

Tray-Steuerung unter Windows

Das Rechtsklickmenü des Tray-Symbols bietet zuverlässig Oberfläche öffnen, Server starten, Server stoppen und Beenden. Server stoppen beendet nur den Survey-Dateiserver. Beenden schließt auch Browseroberfläche und Tray vollständig.

Relative Pfade im Planner auswählen

Der Planner zeigt die vom laufenden Server gefundenen HiPS-Ordner in einem eigenen Dialog mit verständlichen Ordnernamen und dem Survey-Hauptordner an. Eine Nummerneingabe oder ein Browser-Prompt ist nicht mehr erforderlich.
