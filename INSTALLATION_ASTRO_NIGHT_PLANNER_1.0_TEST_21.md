# Installation Astro Night Planner 1.0 Test 21

Diese ZIP-Datei ist die Testversion 21 für das Test-Repository.

## Installation

1. Inhalt der ZIP-Datei vollständig in das Test-Repository übernehmen.
2. Commit und Push ausführen.
3. GitHub Pages Deployment abwarten.
4. Installierte PWA beziehungsweise Browserseite vollständig neu laden und den Update-Hinweis bestätigen.

## Wichtiger Hinweis

Die Service-Worker-Cache-Version wurde auf `1.0.0-test.21` erhöht. Alte Test-Caches werden beim Aktivieren des neuen Service Workers entfernt.
