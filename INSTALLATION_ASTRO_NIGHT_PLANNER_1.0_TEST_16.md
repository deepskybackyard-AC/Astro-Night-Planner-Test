# Installation Astro Night Planner 1.0.0-test.16

Diese Testversion ersetzt die vorherige Testversion vollständig.

## GitHub Pages Testrepository

1. ZIP-Datei entpacken.
2. Den kompletten Inhalt in das Testrepository hochladen bzw. per Git committen.
3. GitHub Actions / Pages-Deployment abwarten.
4. Danach die App mit hartem Neuladen öffnen.

Bei PWA-/Service-Worker-Cacheproblemen: App schließen, Seite neu laden oder den Service Worker für die Testdomain entfernen.

## Wichtige Prüfungen

- Version im Infobereich: `1.0.0-test.16`.
- Aladin-Survey-Auswahlliste in der Rahmungsansicht.
- Einstellungen → Zentrale Einstellungen → Anzeige → Aladin-Surveys.
- Manuelles Ergänzen eines HiPS-Surveys.
- DSS2 red wird nativ/monochrom angezeigt und nicht künstlich rot eingefärbt.
- Mondereignisse in Planungsnacht zeigen vor/nach Planungszeitraum sinnvoll an.
- Externer Aladin-Tab: Abstandsmessung zeigt Zahl und Messlinie.
