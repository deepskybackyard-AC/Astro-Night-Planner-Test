# Installation Astro Night Planner 1.0 Test 22

Diese ZIP-Datei ist die Testversion 22 für das Test-Repository.

## Installation

1. Inhalt der ZIP-Datei vollständig in das Test-Repository kopieren.
2. Vorhandene Dateien überschreiben.
3. Änderungen committen und pushen.
4. Nach dem Deployment die installierte PWA beziehungsweise den Browser-Tab neu laden.
5. Falls noch alte Dateien angezeigt werden, Browser/PWA einmal vollständig schließen und erneut öffnen.

## Cache-Hinweis

Die Service-Worker-Cache-Version wurde auf `1.0.0-test.22` erhöht. Alte Test-Caches werden beim Aktivieren des neuen Service Workers entfernt.
