# Installation Astro Night Planner 1.0.0-test.14

Diese Testversion basiert kumulativ auf 1.0.0-test.13 und korrigiert das Layout der Objektfilter.

## Installation im Test-Repository

1. ZIP-Datei entpacken.
2. Inhalt vollständig in das Test-Repository kopieren.
3. Bestehende Dateien ersetzen.
4. Commit und Push nach `main`.
5. Nach der GitHub-Pages-Veröffentlichung die Seite mit hartem Neuladen öffnen.

## Nach dem Upload prüfen

- Versionsanzeige muss `1.0.0-test.14` zeigen.
- Der Service Worker muss die neue Version laden.
- In der Objektauswahl sollen die oberen Eingabefelder bündig stehen.
- `Max. Magnitude` und `Max. Flächenhelligkeit` sollen gemeinsam im oberen Filterbereich stehen.
- Die Größenfelder bleiben zusammenhängend unter dem Objektgrößenprofil.
