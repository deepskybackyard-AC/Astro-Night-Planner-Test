# Installation Astro Night Planner 1.1.0-test.13

1. ZIP entpacken.
2. Inhalt in das Test-Repository kopieren.
3. Änderungen committen und nach GitHub pushen.
4. Testseite mit PWA-Update oder vollständigem Reload aktualisieren.
5. In Einstellungen → Zentrale Einstellungen → Wetter → Flugwetterstationen Deutschland die Proxy-URL eintragen und mit „Wolkenkarte speichern“ sichern.
