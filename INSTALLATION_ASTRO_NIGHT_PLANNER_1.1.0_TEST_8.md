# Installation Astro Night Planner 1.1.0-test.8

1. ZIP entpacken.
2. Inhalt in das Test-Repository kopieren.
3. Prüfen, dass `.github/workflows/deploy-pages.yml` enthalten ist.
4. Commit und Push ausführen.
5. Nach dem GitHub-Actions-Deploy die Testseite mit STRG+F5 oder dem Update-Button aktualisieren.
