# Astro Night Planner 1.0.0-test.6

Kumulative Testfassung des Astro Night Planners für das separate Test-Repository.

## Wichtig

Dieses Paket ist ausschließlich für das Test-Repository bestimmt. Es enthält alle Korrekturen der vorherigen Teststände; Test 2 bis Test 5 müssen nicht zuvor installiert werden.

Die Anwendung speichert Profile und Einstellungen in IndexedDB. PWA-Dateien liegen in Cache Storage. Vor dem Löschen von Websitedaten immer eine externe Sicherung erstellen.
