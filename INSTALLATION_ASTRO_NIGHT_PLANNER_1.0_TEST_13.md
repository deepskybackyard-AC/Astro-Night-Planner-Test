# Installation Astro Night Planner 1.0.0-test.13

Diese ZIP-Datei ist ausschließlich für das Test-Repository vorgesehen.

## Vorgehen

1. Inhalt der ZIP-Datei vollständig entpacken.
2. Alle Dateien in das Test-Repository `Astro-Night-Planner-Test` kopieren und vorhandene Dateien ersetzen.
3. Änderungen committen und nach GitHub pushen.
4. GitHub Pages neu laden lassen und anschließend im Browser die Testseite hart aktualisieren.
5. Bei installierter PWA gegebenenfalls die App einmal schließen und erneut öffnen.

## Wichtige Prüfung nach dem Upload

- Versionsanzeige muss `1.0.0-test.13` zeigen.
- Service Worker muss die neue Version ausliefern.
- Objektauswahl-Filter müssen mit Beschriftungen oberhalb der Felder dargestellt werden.
- `Meine Aufnahmeziele` muss Such-/Filterfelder, größere Notizen, Speicherbutton und Referenzlink-Öffnen anzeigen.
- Aladin, Open-Meteo, OpenFreeMap und Meteoblue müssen im Livebetrieb erneut geprüft werden.
