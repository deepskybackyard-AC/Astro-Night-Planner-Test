# Installation Astro Night Planner 1.0.0-test.11

Diese Testversion ist kumulativ zur Version 1.0.0-test.9 aufgebaut und enthält zusätzlich die Sprachumschaltung aus test.10 sowie die später vorgemerkten Filter-, Einstellungs- und Aufnahmeziel-Erweiterungen.

## Installation auf GitHub Pages

1. ZIP-Datei entpacken.
2. Inhalt des entpackten Ordners in das Test-Repository kopieren.
3. Änderungen committen und pushen.
4. GitHub Pages Build abwarten.
5. Die PWA im Browser vollständig neu laden. Bei installierter PWA ggf. schließen und erneut öffnen.

## Besonders zu testen

- Planung auf Deutsch und Englisch über `🌐 DE | EN`.
- Objektfilter: Mindestbewertung, Mindestdauer-Basis, Größe Grad/Minuten, Flächenhelligkeit, Aufnahmefilter, Rahmungsfilter.
- Katalogfilter Barnard, LDN/LBN und Meine Aufnahmeziele.
- Einstellungen: Tabs in Zentrale Einstellungen, Standorte & Horizont, Info/Hilfe/Sicherung und Meine Aufnahmeziele.
- Objektlisteninformationen: Kompakt/Standard/Detailliert bleiben beim Verschieben geöffnet.
- Aladin: Mond anzeigen, Objektausrichtung zurücksetzen, Rahmen auf Bildmitte.
- Hilfe und PDF-Handbuch in Deutsch und Englisch.

## Hinweis

Externe Live-Dienste wie Open-Meteo, OpenFreeMap, Meteoblue und Aladin Lite müssen nach dem Hochladen auf GitHub Pages im echten Betrieb geprüft werden.
