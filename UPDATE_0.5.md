# Update 0.5

Dieses Update ergänzt:

- gemeinsame Uhrzeitauswahl für Horizontansicht und Höhenkurve
- Uhrzeitfeld, Nacht-Regler und ±15-Minuten-Schaltflächen
- automatische Aktualisierung von Höhe und Azimut
- markierte gewählte Uhrzeit in der Höhenkurve
- schematische lokale Horizontansicht mit standardmäßig sichtbarem Boden
- Schalter zum Ein-/Ausblenden von Boden und Horizont
- Uhrzeit- und Höhenanzeige über dem Aladin-Himmelsbild
- Meteoblue standardmäßig eingeklappt
- Meteoblue wird bei einem Standortwechsel neu auf den ausgewählten Ort gesetzt
- manuelle Orte werden möglichst einer GeoNames-Ortskennung für Meteoblue zugeordnet

## GitHub aktualisieren

Den Inhalt des Update-ZIP entpacken und über **Add file → Upload files** in das vorhandene Repository laden. Vorhandene Dateien dürfen ersetzt werden. Danach **Commit changes** wählen. GitHub Actions startet den Build automatisch.
