# Installation Astro Night Planner 1.0.0-test.19

1. ZIP-Datei entpacken.
2. Inhalt vollständig in das Test-Repository `Astro-Night-Planner-Test` hochladen.
3. Nach dem Upload im Browser die Testseite öffnen.
4. Bei installierter PWA oder hartnäckigem Cache einmal hart neu laden bzw. die PWA schließen und erneut öffnen.
5. Unter Info prüfen, dass Version `1.0.0-test.19` angezeigt wird.

Wichtig: Für diese Testversion sollten besonders Standortverwaltung, Basisfilter-Layout, Aladin-Objektpopup, Umrisszeichnung und Survey-Einstellungen geprüft werden.
